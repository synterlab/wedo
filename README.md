# WEDO

> Discover emerging assets through chaos.

An experimental onchain discovery game where users drop an energy sphere through a futuristic neural network and discover assets from the Orynth ecosystem.

## Live

[wedo.wtf](https://wedo.wtf)

## Tech

- Pure HTML + Vanilla JS
- HTML Canvas API (neural network board + physics)
- Tailwind CSS CDN
- Space Grotesk + Inter fonts

## Features

- 🎮 Neural network game board with crystal pegs and energy sphere physics
- 💎 Discovery engine with 10 Orynth assets and 6 rarity tiers
- 📊 Live activity feed with real-time updates
- 🏆 Leaderboard, Inventory, Profile tabs
- 📱 Mobile-first, 60fps animations
- 🎨 Premium glassmorphism UI inspired by Apple Vision Pro x Arc x Linear
